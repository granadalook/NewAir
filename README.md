# NewAir

La empresa NEW AIR necesita una solución web para poder conectar viajes a través del
mundo, esta solución debe recibir como parámetros el origen y el destino de viaje del usuario,
después el sistema debe consultar todos los vuelos asociados que tenga, y devolver la ruta de viaje
al usuario si la ruta es posible o un mensaje avisando que la ruta no puede ser calculada

### Installation

1. Clone el repositorio

   ```sh
   git clone https://github.com/granadalook/NewAir.git
   ```

2. Instale dependencias usando NPM

   ```sh
   npm install
   ```

3. Correr el siguiente comando para levantar el proyecto en un servidor local

   ```sh
   ng s -o
   ```

   Este se abrira en su navegador predeterminado en el puerto:4200

4. Para correr los test de la aplicacion por favor use el siguiente comando

   ```sh
   npm run test
   ```

5. Para correr los test y ver la covertura alcanzada por favor use el siguiente comando

   ```sh
   npm run coverage
   ```
   Si desea  ver  la covertura con su interfaz grafica  recuerda abir el `index.html` que se encuentra en la carpeta coverage
